Projects

These projects demonstrate my experience building software using object-oriented programming, graphical interfaces, and problem solving. Each project was developed through coursework and hands-on programming practice.

Road Rage Simulation

A Java-based simulation that models interactions between vehicles and pedestrians on a grid-based map. Different vehicle types follow unique movement rules and behavior patterns.

Technologies: Java, OOP, Interfaces, JUnit

Java Login System

A graphical login interface built with Java Swing that allows users to create accounts, validate passwords, and select campus locations. The application demonstrates GUI design and input validation.

Technologies: Java, Swing GUI

RGB Color Picker Application

A Java GUI application that allows users to dynamically adjust red, green, and blue values using sliders to generate custom colors. The program updates the preview color in real time.

Technologies: Java, Swing, Event Handling

Craps Game Simulator

A Java-based dice game simulation implementing the rules of the casino game Craps. The application includes betting functionality, dice rolling logic, and game state management.

Technologies: Java, Object-Oriented Programming
